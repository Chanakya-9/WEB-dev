# Simon Says Game 🎮

A browser-based Simon Says memory game built with vanilla JavaScript, HTML and CSS.

## 🎯 How to Play
- Click **Start Game** to begin
- Watch the colour sequence flash
- Repeat the sequence by clicking the coloured buttons
- Each level adds one more colour to the sequence
- Game ends when you click the wrong colour

## 🛠️ Built With
- HTML5
- CSS3
- Vanilla JavaScript (no frameworks)

## ✨ Features
- Increasing difficulty each level
- Visual feedback on button press
- Game over screen with final score
- Restart functionality

## 🚀 Live Demo
Open `game.html` directly in any browser — no setup needed!
